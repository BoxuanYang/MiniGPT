=====训练shakespear text的工作流=================

数据预处理, 生成train.bin和val.bin:
python data/shakespeare_char/prepare.py

训练模型:
CUDA_VISIBLE_DEVICES=7 python train.py config/train_shakespeare_char.py --compile=False --batch_size=32


产生样本:
python sample.py --out_dir=out-shakespeare-char

=====rocmstories text的工作流=================

数据预处理, 生成train.bin和val.bin:
python data/rocmstories/prepare.py

训练模型:
CUDA_VISIBLE_DEVICES=1 python train.py config/train_rocmstories.py --compile=False --batch_size=32

评估模型:
python eval_w_endoftext.py   \
 --init_from=resume   \
 --out_dir=out-rocmstories   \
 --input_file=data/rocmstories/test.txt   \
 --device=cuda

产生样本:
python sample_batch.py --init_from=resume --out_dir=out-rocmstories --device=cpu
